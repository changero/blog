  • chrome 多平台各版本下载 (opens new window)

这里记录一下常用的 chrome 扩展，免得经常性的到处找。以下链接都是 google 商店地址，如果没有梯子，也可以到crx4chrome (opens new window)、cnplugins (opens new window)、chromecj (opens new window)、chrome666 (opens new window)、extfans (opens new window)等商店上去寻找

  • toby (opens new window)类似书签管理
  • SwitchyOmega (opens new window)好用的代理工具。github wiki (opens new window)

规则列表：https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt

  • Tampermonkey (opens new window)谁用谁知道的插件，通过脚本扩展目标网站的功能。脚本的下载网址：greasyfork (opens new window)、国外 (opens new window)。推荐脚本 (opens new window),可以破解百度网盘的分享

  • bitwarden，好用的密码管理工具，还可以使用 docker 自建服务

  • globalspeed，视频速度管理，百度云可用

  • stylish，修改网页的 CSS，可以破解 CSDN 的复制

  • octotree (opens new window)浏览 github 仓库的时候再也不用一层一层的去找文件了

  • 广告终结者 (opens new window)亲测拦截优酷广告有效

  • loom (opens new window)浏览器录屏扩展

  • youtube 广告拦截 (opens new window)见明知意

  • mega (opens new window)免费的 50G 云盘

  • octolinker (opens new window)查看 github 仓库代码的时候可以很方便的跳转到其依赖的库

  • npmhub (opens new window)扫描出package.json文件中的依赖，显示在README下面

  • sourcegraph (opens new window)打开一个 github 项目会出现一个图标，点击图标就可以已编辑器的形式打开整个项目

  • SimpleExtManager (opens new window)，便捷的管理已有的扩展（最新版 chrome 已支持类似的功能）

  • bitwarden (opens new window)，免费的密码管理器，支持 APP。

  • 草料 (opens new window)，二维码助手
