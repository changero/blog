### 注意事项

ios端使用`cocoapods`来管理依赖包，

- 安装pod

`cocoapods`使用ruby开发，所以可以用gem安装

> sudo gem install cocoapods

或者使用homebrew安装

> brew install cocoapods # ALL_PROXY=socks5://127.0.0.1:1080 brew install cocoapods
