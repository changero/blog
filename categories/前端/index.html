    Date对象时区转化
    var date = new Date()
    1

    Date对象创建的都是当前时区的时间对象、但是同一时间不同时区创建的时间对象的时间戳是一样的

    1、获取当前时区

        // 单位：小时
    const offset = date.getTimeoutOffset()/60
    1
    2

    offset的值表示格林威治时间与当前时区的时间差值，因此东区是负数，因为东区的时间再格林威治时间之前，比如东八区的offset值是-8
